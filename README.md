# Vextex — Smart Outfit AI

RimWorld mod that improves colonist outfit decisions: role-aware (melee/ranged), thermal safety, DLC support (Royalty, Ideology, Biotech), and compatibility with Combat Extended and modded gear (tags/stats).

**Version:** 1.0.0 (release candidate)
